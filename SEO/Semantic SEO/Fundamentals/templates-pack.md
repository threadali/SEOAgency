# TEMPLATE 2 — Master Keyword Spreadsheet

**Produced in:** Module 2
**Used by:** Module 5 (Mind Map), Module 6 (CMap), Module 7 (Topical Map), Module 8 (Briefs)
**One per:** Central Entity / project sprint

---

**Column headers (copy into spreadsheet row 1):**

```
A: Keyword
B: Monthly Search Volume
C: Keyword Difficulty (0–100)
D: Primary Intent [Informational / Commercial / Transactional / Navigational]
E: Sub-Intent [Definition / Process / Comparison / Troubleshooting / Ranking / Review / Feature / Download / Version / Platform]
F: Entity
G: Attribute
H: Query Path Stage [Awareness / Education / Evaluation / Comparison / Validation / Transaction / Post-purchase]
I: Source [Autocomplete / PAA / Related Searches / Competitor URLs / Tool]
J: Group ID [assign same number to keywords belonging on same page]
K: Page Type [Blog Article / Category Page / Pillar Page / Landing Page / Product Page]
L: Priority [High / Medium / Low]
M: Cannibalism Flag [Y / N — Y = potential conflict with another group]
N: Notes [SERP features, seasonal signals, PAA presence, featured snippet holder]
```

**Sorting and filtering instructions:**
- Primary sort: Column J (Group ID) ascending
- Secondary sort within group: Column B (Volume) descending
- Filter High Priority + Informational → Core article candidates
- Filter High Priority + Commercial → Comparison/review candidates
- Filter Transactional → CMap product/landing page candidates
- Filter Cannibalism Flag = Y → Resolve before building Topical Map

**Quality gate before proceeding to Module 5:**
```
[ ] Every keyword has Primary Intent (Column D)
[ ] Every keyword has Entity + Attribute (Columns F + G)
[ ] Every keyword has Query Path Stage (Column H)
[ ] Every keyword has a Group ID (Column J) — no orphan keywords
[ ] No two groups have identical Intent + Entity + Attribute combinations
[ ] Navigational keywords are separated (will not be used for content planning)
[ ] Total group count = approximate number of Topical Map pages
```

```
T: Author Assignment
U: Brief Status [Not Started / In Progress / Complete]
V: Article Status [Not Started / In Progress / Review / Live]
W: Notes
```

**Intent score values for Column R calculation:**
```
Transactional = 3
Commercial = 2
Informational = 1

Information Gain Bonus:
High (3+ competitor gaps) = +20
Medium (1–2 gaps) = +10
Low (minimal gap) = +0
```

**Topical Map completion quality gate:**
```
[ ] Every article has exactly one Parent CMap Page ID (Column F)
[ ] No articles are orphaned (no parent = structural error)
[ ] URL hierarchy is consistent with CMap URL structure
[ ] No duplicate URLs across CMap + Topical Map combined
[ ] No two articles have identical Macro Context Statements (Column N)
[ ] All root attributes have at least one Core article
[ ] All Mind Map query overlays are covered by at least one article
[ ] All five article sources are represented
[ ] All user groups from Mind Map have at least one corresponding article
[ ] All related entity connections have an intersection article
[ ] Phase classification is correct for all articles
[ ] Every title passes the 5-point title test
[ ] Every article has minimum 2 documented outbound internal link targets
[ ] Every Core article has documented information gain opportunities
```
